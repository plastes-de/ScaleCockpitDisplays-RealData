# ScaleCockpitDisplays-RealData
Interface for Real Data for Scale Cockpit Displays 

In diesem Repository soll das Interface incl. der nötigen Hardware entwicklet werden. Dieses Interface ist Open Source, weil ich nicht der große Hardwarespezialist bin und mir so einen Synergieefeekt mit Kollegen erhoffe, die davon mehr Ahnung haben.

Erläuterung der Pfade:
- RxTxSimulator       --> Sendet Daten im Format der Schnittstelle an den Com-Port.
